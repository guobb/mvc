### mac sudo 

- chmod 755 script.sh可以将script.sh设置为:所者者具有所有权限,组和其他有读和执行权限
- chmod 4755 script.sh,程序将总是作为文件的所有者而执行,只要它不是SHELL脚本就行.文件的所有者具有所有的权限,组和其他人具有读和执行权限. 不管谁 执行该文件,该文件总是能够执行,就好像所有者在执行它一样.
